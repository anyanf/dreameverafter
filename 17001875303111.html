数据结构与算法解析习题2.16：不用递归，写出快速求幂的程序

2019/03/06 posted in  数据结构与算法分析习题

用数组储存，x的1，2，4，8log2（n）次方，其实就是存x的2的数组index次方，储存下来。

然后把n转成2进制，转成2进制后，为1的位数，就是数组的index值，相应的元素，因为是求幂，所以这几个元素相乘。

#include <stdio.h>
#include <math.h>

#define MAXN 1000

int PowersOfXO[MAXN];

int main() {

    int x, n, cnt, i;
    int ans = 1;
    scanf("%d %d", &x, &n);
    cnt = (int)log2(n);
    PowersOfXO[0] = x;
    for (i = 1; i <= cnt; i++)
    {
        PowersOfXO[i] = PowersOfXO[i - 1] * PowersOfXO[i - 1];
    }
    i = 0;
    while (n > 0) {//将n转换为2进制，如果为1乘
        if (n % 2 == 1)
            ans *= PowersOfXO[i];
        i++;
        n /= 2;
    }
    printf("%d", ans);
}
