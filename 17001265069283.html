MAC OS X 常用技巧

2015/04/11 posted in  CS

1. OS X默认是不显示隐藏文件的,想要显示隐藏文件,需要在终端中敲入命令,如下:

  • 显示：defaults write com.apple.finder AppleShowAllFiles -bool true
  • 隐藏：defaults write com.apple.finder AppleShowAllFiles -bool false

对了敲完命令不要忘了重新开启一下finder哦,具体操作如下:

点击屏幕左上角苹果图标 -> 点击强制退出 -> 选择finder ->重新开启

3. zsh切换bash bash切换zsh

// 切换bash
chsh -s /bin/bash

// 切换zsh
chsh -s /bin/zsh

4. Mac OS 10.12 以上安全隐私里没有打开任何来源

输入一下命令即可

sudo spctl --master-disable

5. 打开真机描述文件目录

open Library/MobileDevice/Provisioning\ Profiles/

6. Finder 顶部看见完整的路径

defaults write com.apple.finder _FXShowPosixPathInTitle -bool YES

7. 给指定文件权限

/// 777权限
sudo chmod -R 777 文件名
/// 将指定文件的拥有者改为指定的用户
sudo chown -R $(whoami):admin 文件名
